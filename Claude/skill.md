# YouTube Shorts Creator Skill
 
유튜브 알고리즘과 한국 트렌드를 분석하여 숏츠 주제를 제안하고, 선택된 주제로 완성도 높은
스크립트 + 영상 구성안을 통합 제작하는 스킬.
 
---
 
## 워크플로우 개요
 
```
[PHASE 1] 트렌드 분석 → 5개 주제 추천
      ↓ (사용자가 주제 선택)
[PHASE 2] 스크립트 + 영상 구성안 통합 제작
```
 
---
 
## PHASE 1: 트렌드 분석 및 주제 추천
 
### Step 1-1. 웹 검색으로 현재 트렌드 수집
 
아래 키워드들로 web_search를 실행하여 **오늘 날짜 기준** 최신 트렌드를 파악한다.
각 분야별로 최소 1회 검색할 것:
 
| 분야 | 검색 키워드 예시 |
|------|----------------|
| 밈/유머 | `한국 유튜브 밈 트렌드 2026` |
| 재테크/경제 | `한국 재테크 유튜브 인기 주제 2026` |
| IT/AI | `AI 유튜브 숏츠 트렌드 한국` |
| 라이프스타일 | `한국 라이프스타일 유튜브 인기 2026` |
| 뉴스/이슈 | `한국 유튜브 이슈 급상승 2026` |
 
추가로 유튜브 알고리즘 관련 검색도 수행:
- `유튜브 숏츠 알고리즘 2026 한국`
- `유튜브 급상승 동영상 한국`
### Step 1-2. 5개 주제 선정 기준
 
수집된 트렌드를 바탕으로 아래 기준으로 주제를 평가하여 **서로 다른 분야**에서 5개를 선정한다:
 
- **알고리즘 적합성**: 현재 유튜브가 밀어주는 포맷/주제인가
- **한국 시장 적합성**: 국내 시청자 관심도가 높은가
- **숏츠 제작 가능성**: 3분 이내 세로 영상으로 표현 가능한가
- **신규 채널 유리성**: 대형 채널 대비 진입 장벽이 낮은가
- **반복 제작 가능성**: 시리즈화할 수 있는가
### Step 1-3. 주제 추천 출력 형식
 
아래 형식으로 5개 주제를 출력한다:
 
```
## 🎯 트렌드 분석 기반 숏츠 주제 추천 (YYYY년 MM월 기준)
 
### 📊 현재 유튜브 알고리즘 포인트
[2~3줄로 현재 알고리즘이 선호하는 콘텐츠 특성 요약]
 
---
 
### 주제 1: [제목] | 분야: [분야명]
- **왜 지금인가**: [트렌드 근거 1~2줄]
- **타겟 시청자**: [구체적 타겟]
- **예상 조회수 포텐셜**: 상/중/하 + 이유
- **시리즈 확장성**: [후속 편 아이디어]
- **레퍼런스 채널**: [유사 국내외 채널명]
 
### 주제 2: ...
(이하 동일 형식으로 5개까지)
 
---
> 🔽 원하시는 주제 번호를 선택해주세요. 선택 후 스크립트 + 구성안을 바로 제작해드립니다.
```
 
---
 
## PHASE 2: 스크립트 + 영상 구성안 통합 제작
 
사용자가 주제를 선택하면 즉시 아래 통합 문서를 제작한다.
 
### 출력 구조
 
```
[영상 기본 정보]
[훅(Hook) 설계]
[씬(Scene) 구성안 — 스크립트 + 영상 디렉션 통합]
[자막/텍스트 오버레이 가이드]
[BGM / 사운드 가이드]
[업로드 최적화 정보]
```


## 품질 기준
 
- **대사**: 구어체 자연스러운 한국어, 유튜브 숏츠 말투 사용
- **씬 수**: 총 길이에 따라 4~10개 씬 구성
- **훅**: 반드시 3가지 버전 제공
- **업로드 최적화**: 제목 2개 + 해시태그 필수 포함
- **트렌드 근거**: PHASE 1에서 검색한 실제 데이터를 기반으로 작성
---
 
## 주의사항
 
- 웹 검색 없이 과거 학습 데이터만으로 트렌드를 추측하지 말 것
- 대사는 단순 요약이 아닌 **실제로 말할 수 있는 완성형 문장**으로 작성
- 영상 구성안은 스마트폰 1대로도 촬영 가능한 현실적인 수준으로 작성
- 저작권 이슈가 있는 BGM, 이미지, 영상 클립 사용은 언급하지 말 것
